## Crear proyecto Django
Instalar
 ~~~
pip install django
 ~~~

Crear proyecto
 ~~~
django-admin startproject proyectname .
 ~~~
Run Django
 ~~~
python manage.py runserver
 ~~~
Create app
 ~~~
manage.py startapp appname
 ~~~
Add app in setting.py
